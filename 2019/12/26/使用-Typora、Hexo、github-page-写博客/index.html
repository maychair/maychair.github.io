Hexo 安装与使用

Hexo 是一种博客的管理发布系统，使用 Hexo 可以方便的管理和发布博客；安装步骤如下：

第一步：安装 node.js

node.js 官网下载编译好的安装包，*.tar.xz 格式，使用命令 xz -d node..tar.xz 和 tar -xvf node..tar 解压；设置好环境变量即可使用，export PATH=$PATH:~/you/path/to/nodejs/bin

第二步：新建博客目录，并安装 Hexo

mkdir blog && cd blog && npm install -g hexo

npm install hexo-deployer-git --save –安装 hexo 发布助手

使用 Typora 编辑 markdown 文档

typora 是一款所见即所得的 markdown 文档编辑器；

使用 hexo new "title" 生成一个新的待发布文档，然后使用 typora 编辑 title.md 即可；

注意：typora 保存本地图片的默认路径是本地绝对路径，将文档上传到 github 之后，本地绝对路径是无法识别的，因为你的图片在本地电脑上；需要将图片保存到相对路径，随文档一起上传 github 方可；
